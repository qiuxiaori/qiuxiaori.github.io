### 一. 快捷键

##### Tab

* command + 数字: 切换到指定tab
* command + <-/->: 切换左右tab

##### Pane

* command + d: 垂直切分 pane
* command + shift + d: 水平切分 pane
* command + shift + 上/下/左/右: 切换当前 tab 的 pane
* command + shift + enter: 全屏显示单个pane，再次按下取消全屏

##### iterm

* command + enter: iterm全屏，再次按下取消全屏
